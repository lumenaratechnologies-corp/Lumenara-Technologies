# Lumenara Tech – Neon Gaming-Style Site

A one-page React (Vite) site with Tailwind, Framer Motion and GSAP-inspired effects.

## Scripts
- dev: vite (requires Node 18+)
- build: tsc -b && vite build
- preview: vite preview

## Setup
1. Install Node.js 18+.
2. `npm install`
3. `npm run dev`

## Notes
- Tailwind theme extends a neon palette and glow utilities.
- Smooth scrolling via Lenis; fallback-safe if unavailable.
- Particle canvas in Hero is lightweight and capped for performance.
- Accessibility: focus-visible, semantic landmarks, ARIA on accordions.
- SEO: meta tags + Organization JSON-LD.

## Replace Assets
- Put your logo at `src/assets/logo.svg`.
- Update OpenGraph image at `/og.jpg`.
